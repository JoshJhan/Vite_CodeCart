# 利用vite來實作vue前端框架
> 以購物車實例來練習 *生命週期* 與 *組件之間數據的傳遞*
1. 掌握 **watch** 監聽器的基本使用
    - 定義最基本的 **watch** 監聽器
    - **immediate、deep** 監聽對象中酖個屬性的變化
2. 能夠知道 vue 中常用的生命週期涵數
    - 創建階段、執行階段、銷毀階段
    - **created、mounted**
3. 能夠知道如何實現組件之間的數據傳遞
    - 父子組件、兄弟組件、後代組件
4. 能夠知道如何在 vue3 的項目中全局配置 **axios**
    - main.js 入口文件中進行配置
    - app.config.globalProperties.$http = axios
